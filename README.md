# Align UI: Personal CSS Library
**Align UI** is a lightweight, foundational CSS library built for consistency, modern interface design principles, and a human-centered approach to user experience.

The goal of Align is to provide a standardized set of reusable components and styles across all my personal web projects, ensuring visual harmony and clear structure.

# Installation
```html
<!-- jsDelivr -->
<link rel="stylesheet" href="https://cdn.jsdelivr.net/gh/mowwis/align-ui@latest/dist/align.min.css">

<!-- GitHub Pages -->
<link rel="stylesheet" href="https://mowwis.github.io/align-ui/dist/align.min.css">

<!-- GitHub Pages (unbundled) -->
<link rel="stylesheet" href="https://mowwis.github.io/align-ui/src/main.css">
```
